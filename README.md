# Excel-Depreciation-Calculator

## Project Overview

This project is an Excel-based Depreciation Calculator developed to calculate and compare asset depreciation using two commonly used accounting methods:

* Straight-Line Method (SLM)
* Diminishing Balance Method (DBM)

The project demonstrates how depreciation affects the book value of an asset over its useful life and provides a comparative analysis of both methods.

## Objectives

* Understand depreciation concepts.
* Calculate depreciation using Excel formulas.
* Compare Straight-Line and Diminishing Balance methods.
* Visualize depreciation trends using charts.

## Features

* Automated depreciation calculations.
* Year-wise depreciation schedule.
* Book value tracking.
* Comparison of depreciation methods.
* Graphical visualization using Excel charts.

## Tools Used

* Microsoft Excel
* Excel Formulas
* Data Visualization

## Key Insights

* Straight-Line Method provides equal depreciation every year.
* Diminishing Balance Method provides higher depreciation in initial years.
* The choice of method depends on the asset type and business requirements.
